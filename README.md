# TweetSentiment
Automatic tweet sentiment monitoring and analysis for a given hashtag.

## Requirements
- Python (v3.6):
    - tweepy
    - textblob
- R (v3.0+)
    - data.table

More info can be found in an accompanying [blog post](https://mattravenhall.github.io/2018/03/08/Tweet-Sentiment-Analysis.html).
